# i18n-0004: Preferences sync hook

## Summary

Create `useSyncPreferences` hook to bridge DB → Zustand → i18next/next-themes. Wire into `_authenticated.tsx`.

## Changes

1. **`src/hooks/use-sync-preferences.ts`** (new)
   - On mount: if `user.language` exists → `setPreferences` from DB into store (ref-guarded)
   - Effect: sync `language` → `i18n.changeLanguage()` + `document.documentElement.lang`
   - Effect: sync `theme` → `next-themes` `setTheme()`
2. **`src/routes/_authenticated.tsx`** (edit)
   - Add `SyncPreferences` wrapper component that calls hook + renders `<Outlet />`
