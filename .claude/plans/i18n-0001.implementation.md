# i18n-0001: i18n Core Setup

## Summary

Install i18next deps, create init module with bundled resources, 10 placeholder JSON files.

## Changes

1. **Install deps** — `pnpm add i18next react-i18next i18next-browser-languagedetector`
2. **`src/i18n/index.ts`** (new) — i18next init with:
   - Static imports for all 10 locale JSON files
   - `fallbackLng: 'es'`, `defaultNS: 'common'`
   - 5 namespaces: `common`, `orders`, `settings`, `billing`, `errors`
   - `interpolation.escapeValue: false`
   - `i18next-browser-languagedetector` with `order: ['localStorage', 'navigator']`
   - Exports: `SUPPORTED_LANGUAGES`, `Language` type, `LANGUAGE_CONFIG`
3. **10 placeholder JSON files** — `src/i18n/locales/{es,en}/{common,orders,settings,billing,errors}.json` — all `{}`
