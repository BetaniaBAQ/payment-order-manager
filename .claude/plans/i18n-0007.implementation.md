# i18n-0007: Locale-aware formatting utilities

## Summary

Create `src/lib/format.ts` with `useLocale` hook, pure formatting functions, and date-fns locale map.

## Changes

1. **`src/lib/format.ts`** (new)
   - `LOCALE_MAP` — maps Language to Intl locale (`es-CO`, `en-US`)
   - `useLocale()` — hook returning Intl locale string
   - `formatCurrency(amount, currency, locale)` — Intl.NumberFormat currency
   - `formatDate(timestamp, locale, options?)` — short date
   - `formatDateTime(timestamp, locale)` — date + time
   - `formatFileSize(bytes, locale)` — human-readable KB/MB
   - `DATE_FNS_LOCALES` — `{ es, en: enUS }` for react-day-picker
