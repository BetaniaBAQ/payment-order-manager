# PC-0003: Admin Subscriptions Page

## Summary

New route at `/admin/subscriptions` with super admin guard, org table with tier badges, and tier select dropdowns that call `assignTier`.

## Changes

### New: `src/routes/_authenticated/admin/subscriptions.tsx`

- `createFileRoute('/_authenticated/admin/subscriptions')`
- **Loader**: prefetch `checkIsSuperAdmin` → redirect `/dashboard` if false; prefetch `listOrganizationsWithSubscriptions`
- **Component**: Table with columns: Org name, Slug, Current tier (Badge), Tier select
- Tier select: `free`/`pro`/`enterprise` → calls `assignTier` via `useMutationWithToast`
- Get `authKitId` via `authRoute.useLoaderData()` (same pattern as all other authenticated routes)

### Modify: `src/lib/constants/navigation.ts`

- Add `adminSubscriptions: '/admin/subscriptions'` to `ROUTES`

## UI Components Used

- `Table`, `TableHeader`, `TableBody`, `TableHead`, `TableRow`, `TableCell` from `@/components/ui/table`
- `Badge` from `@/components/ui/badge`
- `Select`, `SelectTrigger`, `SelectValue`, `SelectContent`, `SelectItem` from `@/components/ui/select`
