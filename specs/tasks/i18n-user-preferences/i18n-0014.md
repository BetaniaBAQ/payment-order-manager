## Requirements

1. Add `useTranslation()` with appropriate namespace per page
2. Translate:
   - Page titles and headings
   - Section headers
   - Empty state messages passed to `EmptyState` component
   - Button labels ("New Profile", "Create Organization", etc.)
   - Breadcrumb labels passed to `AppHeader`
   - Toast messages at `useMutationWithToast` call sites

3. Update `src/lib/constants/navigation.ts`:
   - `BREADCRUMB_LABELS` values become translation keys or remove constant entirely and use `t()` inline

## Definition of Done

- [ ] All route pages use `t()` for user-facing strings
- [ ] Page titles and headings translate
- [ ] Empty states translate
- [ ] Breadcrumbs translate
- [ ] Toast messages translate
- [ ] No hardcoded user-facing strings in any route file
