## Requirements

### `common.json` — shared UI strings

Keys to include:

- `appName`, `appTitle`
- Actions: `save`, `saving`, `cancel`, `delete`, `deleting`, `confirm`, `processing`, `clear`, `search`, `create`, `edit`, `close`, `signOut`, `back`
- Breadcrumbs: `breadcrumbs.settings`, `breadcrumbs.newOrganization`
- Preferences: `preferences.language`, `preferences.theme`, `preferences.light`, `preferences.dark`
- Errors: `errors.somethingWentWrong`, `errors.reloadPage`, `errors.operationFailed`, `errors.notFound`
- Empty states: `empty.organizations.*`, `empty.profiles.*`, `empty.paymentOrders.*`
- Roles: `roles.owner`, `roles.admin`, `roles.member`
- Misc: `loading`, `noResults`, `required`, `optional`

### `orders.json` — payment order domain

- `status.*` — all 8 statuses (CREATED through CANCELLED)
- `actions.*` — labels, confirm titles, confirm descriptions for each action
- `info.*` — field labels (amount, reason, description, tag, createdBy, etc.)
- `timeline.*` — action descriptions for history entries
- `filters.*` — filter labels and placeholders
- `documents.*` — document section strings
- `create.*` — create order dialog labels
- `toast.*` — success/error messages for order operations

### `settings.json` — org/profile settings

- `tabs.*` — General, Members, Billing tab labels
- `general.*` — org details, danger zone strings
- `members.*` — table headers, invite strings, role labels
- `profile.*` — profile settings labels
- `toast.*` — success/error messages for settings operations

### `billing.json` — billing/pricing

- `tiers.*` — tier names and feature descriptions
- `pricing.*` — pricing page strings
- `subscription.*` — subscription status, billing cycle strings
- `usage.*` — usage meter labels
- `limits.*` — limit banner strings
- `toast.*` — billing toast messages

### `errors.json` — validation/error messages

- `validation.*` — zod error messages (required, min, max, email, positive, etc.)
- `auth.*` — auth-related errors
- `network.*` — generic network errors
