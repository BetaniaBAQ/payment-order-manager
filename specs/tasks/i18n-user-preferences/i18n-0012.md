# 0012: Translate organization settings components

## Context

Organization settings has 5 components with hardcoded English strings for tabs, table headers, form labels, roles, and CRUD actions.

## Dependencies

- 0006 (i18n wired)
- 0008, 0009 (translations)

## Files

- `src/components/organization-settings/general-settings.tsx` (edit)
- `src/components/organization-settings/members-settings.tsx` (edit)
- `src/components/organization-settings/member-row.tsx` (edit)
- `src/components/organization-settings/invite-dialog.tsx` (edit)
- `src/components/organization-settings/invite-row.tsx` (edit)

## Requirements

1. Add `useTranslation('settings')` to each component
2. Translate:
   - Tab labels: "General", "Members", "Billing"
   - Section headers: "Organization Details", "Danger Zone"
   - Form labels and placeholders
   - Table headers: "Name", "Email", "Role", "Actions"
   - Role labels: "Owner", "Admin", "Member"
   - Button labels: "Invite", "Remove", "Change role"
   - Confirmation dialogs
   - Toast messages (use `t()` at `useMutationWithToast` call sites)

3. If role constants exist as a separate map, convert labels to translation keys

## Definition of Done

- [ ] All 5 organization settings components use `t()` for strings
- [ ] Tab labels, form labels, table headers all translatable
- [ ] Role labels translate correctly
- [ ] Toast messages translate correctly
- [ ] No hardcoded user-facing strings remain
