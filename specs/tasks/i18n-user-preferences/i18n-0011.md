## Requirements

### Constants migration

`STATUS_CONFIG` in `src/constants/payment-orders.ts`:

- Change `label: string` to `labelKey: string` with translation key values
- E.g. `CREATED: { labelKey: 'orders:status.created', ... }`
- Components resolve: `t(config.labelKey)`

`ACTION_CONFIG` in `order-actions.tsx` (if object map exists):

- Same pattern: store keys, resolve at render with `t()`

### Formatting

Replace all local `formatDate`/`formatCurrency` functions:

- Import `useLocale`, `formatCurrency`, `formatDate`, `formatDateTime` from `@/lib/format`
- Use `useLocale()` hook in component, pass locale to formatter functions

### Key components

**`order-actions.tsx`** — most complex: ~30+ strings in labels, confirm dialogs, descriptions
**`order-timeline.tsx`** — interpolated strings like "changed status from {{from}} to {{to}}"
**`order-filters.tsx`** — placeholder texts, "Clear", "All statuses", filter labels
**`create-order-dialog.tsx`** — form labels, placeholders, validation messages
**`documents-list.tsx`** — "No documents", "Delete document?", file size display
