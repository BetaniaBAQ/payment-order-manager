## Requirements

1. Add `useTranslation('billing')` to each component
2. Extract all existing Spanish strings to `billing.json` translation keys
3. Replace hardcoded `Intl.DateTimeFormat('es-CO', ...)` with locale-aware `formatDate()`
4. Move display-only data from `convex/lib/tierLimits.ts`:
   - `TIER_LABELS` → `billing:tiers.{tierName}.name` in translation files
   - `TIER_FEATURES` → `billing:tiers.{tierName}.features` array in translation files
   - Keep `TIER_LIMITS` (numbers/thresholds) server-side
